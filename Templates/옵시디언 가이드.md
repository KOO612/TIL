### 3단계: 기본 화면 구성 파악

```
┌─────────┬──────────────────┬─────────────┐
│         │                  │             │
│  파일   │   에디터 영역    │  우측 패널  │
│  탐색기 │  (노트 작성)     │  (백링크 등)│
│         │                  │             │
│         │                  │             │
└─────────┴──────────────────┴─────────────┘
```

- **좌측**: 파일 탐색기, 검색, 태그 목록
- **중앙**: 노트 작성 공간
- **우측**: 백링크, 아웃라인 등
- **상단 리본(왼쪽 세로줄)**: 주요 기능 아이콘들

### 7단계: 핵심 기능 3가지 먼저 익히기

#### ① 양방향 링크 `[[]]`

노트 작성 중 `[[` 입력하면 노트 검색창이 뜸

```
예: JWT를 공부하다가 [[OAuth]] 개념이 나오면
→ [[OAuth]] 라고 쓰면 OAuth 노트로 연결됨
→ OAuth 노트가 없으면 자동으로 새로 생성됨
```

#### ② 빠른 전환 `Ctrl+O`

파일 이름 검색해서 바로 이동. 매우 자주 씀.

#### ③ 커맨드 팔레트 `Ctrl+P`

모든 기능을 검색해서 실행. 단축키 모를 때 여기서 찾으면 됨.
